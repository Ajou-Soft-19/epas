 # [EPAS(Emergency vehicle Pre-Alerting System)]
---
## 프로젝트 소개
- EPAS는 차량 주행자가 응급차량의 접근을 미리 알 수 있습니다.
- 응급차량은 응급상황에서 보다 쉽고 빠르게 목적지에 도달할 수 있습니다.
- 주행자는 응급차량의 접근을 보고 받아 사고 위험없이 미리 길을 터줄 수 있습니다.
